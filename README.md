# Desi-ASMR
<i>Desi-ASMR, an audio ASMR app with a focus on Hindi and other Indian languages.</i>

<h2> Hey Everyone,</h2>
I am planning to build an app for ASMR audios, especially hindi and other Indian languages. 
It lets you upload, record, share high quality ASMR audios. Here the user can either listen to audios or become a ASMR artist themselves.
A vibrant community with local language support will be given priority. With extensive search options, recommendation systems, explicit user and AI tagging, it helps find the right content for the users.
The script sharing features shall also be considered.
